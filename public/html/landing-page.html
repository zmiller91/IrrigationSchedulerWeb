How It's Done

You use our website to communicate to our web servers, we use RabbitMQ to communicate to your Raspberry Pi.

Each time you modify a schedule or perform an action we send a message to your Raspberry Pi. For every message, your Raspberry Pi processes the message and performs an operation. You can add, delete, edit, stop, or start irrigation schedules.

You tell us what you want your Pi to do and we tell your Pi to do it.

Raspberry Pi Instillation and Configuration

Most of the heavy lifting is already done, all that's needed is installation and configuration. These instructions operate under the assumption that you have the following:

  1. A Raspberry Pi with an Internet connection
  2. Python 2.7 installed
  3. Git installed

Install Dependencies

RabbitMQ and Python-Crontab


$ sudo apt-get update
$ sudo apt-get install rabbitmq-server
$ sudo apt-get install python-crontab

Download pip if you don't already have it. You have pip if you get the following:


$ which pip
/usr/local/bin/pip

If you don't have it then you can install it.


$ wget https://bootstrap.pypa.io/get-pip.py
$ sudo python get-pip.py

Pika and GPIO


$ sudo pip install pika
$ sudo apt-get install python-rpi.gpio

Download LawnTab


$ cd && git clone https://github.com/zmiller91/LawnCronPi.git
