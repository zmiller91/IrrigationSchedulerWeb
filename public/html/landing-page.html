How It's Done

You use our website to communicate to our web servers, we use RabbitMQ to communicate to your Raspberry Pi.

Each time you modify a schedule or perform an action we send a message to your Raspberry Pi. For every message, your Raspberry Pi processes the message and performs an operation. You can add, delete, edit, stop, or start irrigation schedules.

You tell us what you want your Pi to do and we tell your Pi to do it.

LawnTab Installation and Configuration

Most of the heavy lifting is already done, all that's needed is installation and configuration. These instructions operate under the assumption that you have the following:

  1. A Raspberry Pi with an Internet connection
  2. Python 2.7 installed
  3. Git installed

Install Dependencies


$ sudo apt-get update
$ sudo apt-get install rabbitmq-server
$ sudo apt-get install python-pika
$ sudo apt-get install python-crontab
$ sudo apt-get install python-rpi.gpio

Download


$ cd && git clone https://github.com/zmiller91/LawnCronPi.git

Configure

All configuration parameters are located in configuration.py. There are three main configuration parameters that need to be set:

  • python: the location of your python executable, or the output of `which python`.
  • driver: the absolute path of valve_driver.py
  • id: the id of your Raspberry Pi. These are generated at lawncron.com/rpis.

All other configurations are optional. Here's why our configuratino file looks like: 


import logging

# LawnTab configuration
id = "6ec32790"
python = "/usr/bin/python" # output of `which python`
driver = "/home/pi/LawnCronPi/valve_driver.py"

# RabbitMQ host
rmq_host = "lawncron.com"

# Logging
log_file="/tmp/lclog"
log_level = logging.INFO

# Job execution
pid_files = "/tmp/lcpids"
cleanup_frequency = 60.0  # in seconds

Building the Circuit

There are four zones that can be turned on or off. Each zone has a LED, mosfet transistor, and a terminal block. The circuit also contains eight resistors and four diodes. Each zone is connected to a GPIO pin and a 12v power supply. When the GPIO pin reads HIGH the transistor opens and the 12v flows to the terminal block.
